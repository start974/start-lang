# Semantic

```
------------------ CONSTANT
  Δ ⊢ c --> V(c)


 Δ[x] = e    Δ ⊢ e ---> Δ ⊢ v
------------------------------- VAR
  Δ ⊢ x ---> Δ ⊢ v


-------------------------------- DEFINITION
  Δ ⊢ def x = e |--> Δ, x = e


  Δ ⊢ def x2 = e2 ... def xn = en |--> Δn
---------------------------------------------------- PROGRAM
  Δ ⊢ def x1 = e1 ... def xn = en |--> Δn[x1 = e1]
```
